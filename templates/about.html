About

Usage

Paste a game log of a Secret Hitler game into the input box, and click predict!  Make sure your input follows the correct formatting.
The fun comes in when you input partial games -- the model can run on these as well and give you live predictions as to who is who in your game!
                    
The outputs are probabilities of each player being a Fascist, where 0.00 represents a certain Liberal and 1.00 represents a certain Fascist.         

Model

For more information on the machine learning model and how it works, check out the Deep Red Project.
